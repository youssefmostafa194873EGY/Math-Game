🧮 Math Quiz Game (C++)

A console-based math quiz game built with C++. The application generates random arithmetic questions based on user-selected difficulty and operation type, allowing players to test and improve their math skills interactively.

🚀 Features

Custom number of questions

Multiple difficulty levels:

Easy (1–10)

Medium (10–100)

Hard (100–1000)

Mixed

Supported operations:

Addition (+)

Subtraction (−)

Multiplication (×)

Division (÷)

Mixed mode

Instant feedback on answers

Final performance summary (Pass/Fail)

Replay option

🧠 Concepts Used

struct for data modeling

enum for operation and answer types

Modular function design

Random number generation

Input validation

Basic game loop logic

▶️ How to Run

Compile the code:

g++ main.cpp -o math_game

Run the program:

./math_game
🎮 Gameplay

Choose:

Number of questions

Difficulty level

Operation type

Solve each question

Get immediate feedback (correct/incorrect)

View final results at the end
